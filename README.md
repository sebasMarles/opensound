# OpenSound

## Objetivo
La app busca ser una propuesta para bandas emergentes que suban sus canciones sin licenciamiento y para personas que tengan el incentivo de escuchar música fuera de lo mainstream y descubrir nuevos gustos. Esto ayuda a las bandas emergentes debido a que subir música sin licenciamiento es grátis, sin embargo, en las apps de streaming de música principales, esto cuesta tanto para el consumidor como para el usuario que desee subir su música


---

## Características principales

- Interfaz simple y optimizada para dispositivos móviles.  
- Modo oscuro para una experiencia visual agradable.  
- Búsqueda de canciones, álbumes y listas de reproducción.  
- Navegación clara mediante pestañas y pilas de navegación.  
- Sección de actividad reciente para acceder rápidamente a la música reproducida.
